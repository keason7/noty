## Install

### Clone the project
```
git clone [repo_url]
cd noty
```

### Rename .env file
```
mv .template_env .env
```

### Edit preferences
root_path = path to dir where noty is installed  
text_editor = favorite text editor such as gedit (keyword=**gedit**), sublime-text (keyword=**subl**)

```
root_path=/path/to/install_dir
text_editor=gedit
```
